### Exporting mechanics

- **Distinguish direct cost vs absorption (absorbent) cost** — including when each is used and how direct-cost export pricing can expose a firm to dumping accusations. → [T05](topics/T05%20-%20Export%20Costing.md)
- **Devise an export plan using the 4Ps framework** (Product / Promotion / Price / Physical distribution) — essay-length answer, ideally using the Baja California wine exercise as a template. → [T06](topics/T06%20-%20Export%20Plan%204Ps.md)
- **Name the 5 documents required in every international trade transaction** and state who prepares each. → [T07](topics/T07%20-%20Export%20Documents%20and%20Flow.md)
- **State the 3 purposes of the Bill of Lading** (document of legal possession; contract; delivery receipt). → [T07](topics/T07%20-%20Export%20Documents%20and%20Flow.md)
- **Describe the 8-step flow of a trade transaction** (negotiation → customs agent → Pedimento → Cert of Origin/Packing/Invoice → Bill of Lading → ship → duties paid → customs inspection). → [T07](topics/T07%20-%20Export%20Documents%20and%20Flow.md)
- **Explain the 8-digit Harmonized Customs Code (HCC)** and the role of the customs broker. → [T07](topics/T07%20-%20Export%20Documents%20and%20Flow.md)
